# scrapy book
https://bloogle.top/ blog crawler, crawl e-book.

## 介绍

爬取博客电子书，按照博客的书单标签建立目录，下载当前标签下的电子书。

## 准备

- `python3.5.2+`
- `scrapy`

## 开始

如果没有安装`scrapy`需要先安装爬虫框架

### 安装环境

```shell
# 利用pip安装scrapy
pip install scrapy
```

如果没有错误安装成功后，即可开始下面的工作。

### 创建项目

```shell
cd worksapce
D:\worksapce>scrapy startproject scrapy_book
```
